# Redesign Concept — Laleh Tehran Hotel

یک وب‌سایت چندصفحه‌ای مدرن برای هتل بین‌المللی لاله تهران با React، Three.js و Framer Motion.

## اجرای سریع

```bash
cd laleh-tehran-redesign
npm install
npm run dev
```

برای خروجی پروداکشن:

```bash
npm run build
npm run preview
```

خروجی قابل مشاهده داخل Arena در مسیر زیر ساخته شده است:

```text
site-preview/index.html
```

## صفحات ساخته‌شده

- خانه با هیرو سینمایی، Three.js، فرم رزرو سریع و سکشن‌های هتل
- اتاق‌ها و سوئیت‌ها با فیلتر/مرتب‌سازی
- صفحه جزئیات مستقل برای هر اتاق
- رزرو آنلاین کامل فرانت‌اند
- رستوران‌ها و رزرو میز
- سالن‌ها و رویدادها + فرم درخواست رویداد
- امکانات رفاهی و تندرستی
- گالری قابل فیلتر با Lightbox
- راهنمای تهران‌گردی اطراف هتل
- درباره ما
- تماس با ما + فرم پیام
- صفحه 404

## بخش‌های واقعاً کارکردی

- موتور رزرو: اعتبارسنجی تاریخ، تعداد شب، ظرفیت اتاق، خدمات اضافه، محاسبه مبلغ، مالیات/خدمات، ثبت رزرو و صدور کد پیگیری.
- فرم رزرو رستوران: ذخیره درخواست در LocalStorage.
- فرم رزرو سالن: ذخیره درخواست در LocalStorage.
- فرم تماس: ذخیره پیام در LocalStorage.
- فیلتر اتاق‌ها، مرتب‌سازی، گالری، لایت‌باکس و ناوبری چندصفحه‌ای.

> نکته: این نسخه بدون بک‌اند است. برای لانچ واقعی باید موتور رزرو به API موجودی اتاق، درگاه پرداخت، CRM و سیستم پیامک/ایمیل هتل وصل شود.

## تکنولوژی‌ها

- React + Vite
- React Router
- Three.js برای صحنه سه‌بعدی هیرو
- Framer Motion برای موشن‌ها و ترنزیشن‌ها
- CSS ریسپانسیو و RTL

## منبع محتوا

متن‌ها و نام اتاق‌ها/خدمات از محتوای وب‌سایت فعلی گروه هتل‌های لاله الهام گرفته و بازنویسی/بازطراحی شده‌اند.
